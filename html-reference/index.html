A valid HTML page includes:

  • The start and end tag
  • The <!DOCTYPE html> heading, which defines the HTML file to the browser
  • <!html> and<!/html>, which states that everything in between is in html
  • <!body> and </body>, which states that everything in between is in the body of the website

HTML Element

Description

The HTML start and end tag state that everything in between is in HTML

Syntax

Placed at the beginning and end of the code just before the body element

Example

<html> (code...) </html>
